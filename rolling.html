Installing

This guide will focus specifically on dwm, as it is one of the more involved installs. The installation and patching process is the same for any other Suckless software though.

Before you do any patching and the like, you need to actually install dwm. To do this, clone the repo from the suckless site.

Inside the working directory, all we need to do is to tell the makefile to install the software for us, so run make && sudo make clean install notice that we didn't give sudo to make, it's better practice to not run commands as root unless you absolutely need to.

If the make didn't succeed, chances are you are missing some headers. On my fresh install, I was missing the X11/Xft/Xft.h header and X11/extensions/Xinerama.h header, so you will need to install the libxft and libxinerama packages respectively (for arch at least).

The second issue I had at first was that I didn't have any fonts installed, I got the adobe-source-code-pro-fonts package.

... and that's it! The installation process is done, not so hard right? If you use X, simply add the line: exec dwm to the bottom of you .xinitrc file. Keep in mind that any lines after this will be ignored. Now on your next X server start, you will be greeted with dwm, in all it's beautiful glory.

Basics
