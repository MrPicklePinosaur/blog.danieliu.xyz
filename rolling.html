If you are here, I will assume that you are already convinced on why suckless software is awesome, and are just trying to find out how the heck you use it.

> Installing

Before you do any patching and the like, you need to actually install the software. To do this, clone the repo from the suckless site For this example, I will be demonstrating how to install and patch dwm, the suckless window manager, but the same process applies for any other suckless software. Inside the directory, run make && sudo make clean install
