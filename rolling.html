animelon

Fri, Dec 18 14:31

Great site anime site for learning japanese called animelon that provides english subtitles as well as japanese (with options for romaji and hiragana). You can also hover over subtitles to get definitions, and jump between dialogue lines. Anyways, it has some decently high quality stuff so it can used as a normal anime site. Only downside is that it doesn't have the newer stuff, since they need to get the subs and translate them.
